# E-Commerce Website

A modern and fully responsive E-Commerce web application built 
with React.js that lets users search products, manage quantity, 
add to cart, and track orders — all in a clean Amazon-style UI.

## ✨ Features

-  **Search Bar** — Search products instantly by name
-  **Product Listing** — Grid layout with product image, 
  name, rating, reviews and price
-  **Star Ratings** — Visual rating with review count 
  for each product
-  **Quantity Selector** — Dropdown to choose quantity 
  before adding to cart
-  **Add to Cart** — Add any product to cart with 
  one click
-  **Orders Page** — View your placed orders
-  **Cart Counter** — Live cart item count shown in navbar
-  Fully **Responsive** for mobile and desktop
